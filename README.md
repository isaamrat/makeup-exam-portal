# BRACU CSE Makeup Exam Portal

A clean, focused dashboard for coordinating makeup exams. Built for academic teams who need a fast way to organize schedules, spot conflicts, and keep everyone aligned.

## Highlights
- Visual timeline view of each day’s exam schedule
- Conflict detection to catch overlaps early
- Quick add/edit/remove flow for exams
- Calendar navigation to jump across dates
- Clean, modern UI that works on desktop and mobile

## Who It’s For
- Course coordinators managing makeup exam logistics

## Privacy & Data Handling
This app is designed to keep data within the authorized system.

- **No unauthorized sharing**: User data is not sent to any unauthorized sources.
- **Authorized access only**: Data is stored in the official database and is shared only with approved coordinators.
- **Authentication**: Uses Google Sign-In. The app stores the session token and basic profile details (email/name) in the browser’s local storage to keep you signed in.
- **Retention & control**: You can remove exams you created and log out to clear the stored session token from this device.

## Screens & Flow
- Sign in with your institutional Google account
- Pick a date from the calendar
- View and manage the day’s schedule in the timeline
- Resolve conflicts and adjust slots quickly
